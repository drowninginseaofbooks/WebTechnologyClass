2.2 Document Meta

                Meta tags are used as an important elements by the search engines. They are used to describe HTML 
                elements and communicate the content of a website for search engines. Webstie meta data inclues, 
                page titles, descriptions, keywords and many more to help search engines to crawl and rank web pages.

                Example:
                <meta name="author" content="Rohan Sunuwar">
                <meta name="description" content="test1">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <meta http-equiv="refresh" content="5">
